Python 中 “def” 关键字的目的是什么:它指示函数的开始，以下缩进的代码部分将存储以供以后使用

整数除法对正数和负数都向下舍入。
5 // 3 # => 1
-5 // 3 # => -2
5.0 // 3.0 # => 1.0 # works on floats too
-5.0 // 3.0 # => -2.0

2**3 # => 8 (3次方)

7 % 3 # => 1

i % j have the same sign as j, unlike C

-7 % 3 # => 2

布尔值是基元（注意：大写）
True # => True
not True # => False

注意“and”和“or”区分大小写
True and False # => False(&&)

True 和 False 实际上是 1 和 0，但关键字不同

True 和 False 实际上是 1 和 0，但关键字不同

All other values are True

bool(0) # => False
bool(“”) # => False
bool([]) # => False
bool({}) # => False
bool(()) # => False
bool(set()) # => False
bool(4) # => True
bool(-6) # => True

（is vs. ==） 是检查两个变量是否引用同一个对象，但 == 检查是否指向的对象具有相同的值。

A string can be treated like a list of characters

“Hello world!”[0] # => ‘H’

You can find the length of a string

len(“This is a string”) # => 16

Since Python 3.6, you can use f-strings or formatted string literals(使用 F 字符串或格式化字符串文本).

name = “Reiko”
f”She said her name is {name}.” # => “She said her name is Reiko”

Any valid Python expression inside these braces is returned to the string.

f”{name} is {len(name)} characters long.” # => “Reiko is 5 characters long.”
理解:{}为一个对象,不管里面包什么

None is an object

None # => None

Don’t use the equality “==” symbol to compare objects to None

Use “is” instead. This checks for equality of object identity.

“etc” is None # => False
None is None # => True
